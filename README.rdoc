== Rails OAuth Client

Este é um exemplo simples de integração via OAuth 2.0 com o VPSA.

== Usando o projeto

1. Cadastre seu app no DevCenter da VPSA.

2. Coloque seu APP_ID e seu APP_SECRET em app/controllers/oauth_controller.rb

3. Suba o servidor rails local

4. Teste a integração: link:http://localhost:3000
